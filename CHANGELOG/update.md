# 🔄 Atualização do Projeto: MommyBee

## 📌 Contexto
O aplicativo anteriormente descrito como um marketplace + rede social para mães agora possui um **nome oficial**:

# 🐝 MommyBee

O nome remete à **abelha (Bee)**, símbolo de:
- cuidado
- comunidade
- cooperação
- trabalho coletivo
- aspecto maternal (colmeia / cuidado com a cria)

O branding do app deve refletir esses valores.

---

# 🎯 Objetivo da Atualização

Atualizar o projeto para:

1. Adotar o novo **nome MommyBee**
2. Reestruturar a **Home focada em Marketplace**
3. Criar uma **logo minimalista**
4. Ajustar identidade visual
5. Manter compatibilidade com **Termux e ambiente mobile**

# 🐝 Logo

Criar uma **logo minimalista** baseada no conceito de abelha.

## Requisitos

- estilo **linha preta**
- minimalista
- escalável
- fundo transparente
- formato SVG

## Conceito visual

Combinar:

- uma **abelha simples**
- **elementos de maternidade**
- ou **colmeia / hexágono**

### Ideias possíveis

1️⃣ Abelha minimalista com linhas simples  
2️⃣ Hexágono (colmeia) com letra **M** dentro  
3️⃣ Abelha com asas formando um **coração**  
4️⃣ Ícone simples + texto MommyBee
